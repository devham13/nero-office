# Шаблоны страниц для темы Kadence

При публикации **Юра** загружает в `get_stylesheet_directory()`:

| Файл | Назначение |
| --- | --- |
| `page-{slug}.php` | Шаблон WordPress страницы |
| `longread-page-kadence-layout.css` | Сброс Kadence, один скролл, скрытие `#masthead` |
| `nero-ai-floating-header.css` | Pill-шапка как на главной |
| `nero-ai-floating-header.js` | Меню / smooth scroll |
| `nero-ai-floating-header.inc.php` | HTML шапки (подключается из PHP) |
| `longread-page-design-reference.css` | Тёмный дизайн `.nero-ai-home-page` + лонгрид |
| `longread-page-reveal.js` | Reveal и tooltips |

Источник правды в репозитории: `shared/` (копировать в тему при деплое).

Канонический пример: `page-vnedrenie-ai-obrabotka-zayavok-s-sayta.php`.
