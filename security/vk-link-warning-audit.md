# Аудит: предупреждение ВКонтакте и «двойные» переходы по ссылкам

**Дата:** 8 июня 2026  
**Пример:** ссылка из поста ВК через `vk.com/away.php?to=…` на slug `vnedrenie-ai-obrabotka-zayavok-s-sayta/`  
**Статус сайта:** вредоносного кода и скрытых редиректов не обнаружено

---

## 1. Предупреждение ВКонтакте — что это

Текст «Ссылка может вести на вредоносный сайт…» показывает **сам ВКонтакте** на промежуточной странице `away.php`, **до** перехода на ваш сайт. Это **не антивирус браузера** и **не блокировка хостинга**.

ВК оборачивает **все внешние ссылки** из постов/комментариев через:

```
https://vk.com/away.php?to=<закодированный URL>
```

Поэтому пользователь видит **два шага**:

1. `vk.com/away.php?...` (экран предупреждения ВК)
2. ваш сайт `/vnedrenie-ai-obrabotka-zayavok-s-sayta/` (целевая страница)

Это **нормальное поведение ВК**, а не «взлом» или редирект на стороне сайта.

### Почему ВК может ругаться на легитимный сайт

Эвристики репутации (не полный список):

- молодой домен без накопленной репутации;
- коммерческие лонгриды с брендами Meta / WhatsApp / Яндекс / Microsoft;
- CTA в Telegram;
- ранее опубликованные дубли страниц и черновики с заголовком «hero fix»;
- отсутствие ранее юридических страниц (исправлено 08.06.2026).

**Технически сайт чист:** HTTP 200, SSL валиден, `wp core verify-checksums` OK, нет `eval`/скрытых iframe, нет JS-редиректов при загрузке страниц.

## 4. Рекомендации для постов ВКонтакте

1. **В постах указывайте канонический URL** (slug `vnedrenie-ai-obrabotka-zayavok-s-sayta/`, не короткий дубль).  
   (не старый `ai-obrabotka-zayavok-s-sayta/` — он теперь 301).

2. **Не пугайте формулировками:** «официальный сайт Meta», «войти в аккаунт», «подтвердить данные», «забрать выплату».

3. **Предупреждение `away.php` убрать полностью нельзя** — это политика ВК. Со временем и при росте доверия к домену оно может ослабнуть.

4. **Обратная связь ВК:** [Поддержка ВКонтакте](https://vk.com/support) — если домен ошибочно в чёрном списке, можно запросить пересмотр (укажите ИНН, политику конфиденциальности, контакты).

5. **Повторный аудит:** `python3 scripts/audit-site-redirects.py`
