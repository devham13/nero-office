# Чеклист мобильной проверки (mobile-agent / QA)

Используются **mobile-agent** (обязательно после indexator) и **Макс (QA)** (подтверждение). Эталон: главная `${WP_SITE_URL}`.

## Viewport

Проверять **все три** ширины:

| Ширина | Типичное устройство |
|--------|---------------------|
| 360px | Компактные Android |
| 390px | iPhone 12/13/14 |
| 430px | iPhone Plus / крупные телефоны |

## Первый экран

- [ ] Шапка видна (pill-header темы, не legacy sticky-nav)
- [ ] Нет большой пустой полосы между шапкой и hero (`padding-top` сброшен — см. `agent-pipeline-pitfalls.md` §1)
- [ ] Hero `.nero-ai-hero` отображается
- [ ] H1 читаем, не обрезан, переносится
- [ ] Подзаголовок читаем
- [ ] Главная CTA видна и кликабельна на **360px**
- [ ] Нет наложения pills/tasks/H1
- [ ] Нет горизонтального скролла
- [ ] Первый экран не «уезжает» из-за лишних отступов

## Мобильное меню

- [ ] Меню открывается
- [ ] Пункты кликабельны
- [ ] Якоря ведут к существующим `id`
- [ ] Скролл к секции корректный
- [ ] Меню закрывается штатно
- [ ] После закрытия не перекрывает CTA

## Страница целиком

- [ ] Карточки/сетки — одна колонка на mobile
- [ ] Изображения не вылезают за экран
- [ ] Таблицы/длинные строки без horizontal overflow
- [ ] FAQ аккордеон работает
- [ ] Внутренние ссылки не ломают вёрстку
- [ ] Footer не сломан
- [ ] Нет маркеров старого дизайна (`amo-sticky-nav`, `ym-sticky-nav`)

## Технический CSS (при правках)

- [ ] Нет жёстких `width` > viewport без media query
- [ ] Осторожно с `100vw`
- [ ] Есть `@media (max-width: …)` для hero/H1/CTA
- [ ] `flex-wrap` где нужно
- [ ] `max-width: 100%` у широких медиа

## Статусы

| Статус | Значение |
|--------|----------|
| ✅ ГОТОВО | Все критичные пункты OK на 360/390/430 |
| ⚠️ WARNING | Мелкие замечания, не блокируют QA |
| ❌ БЛОКЕР | Первый экран сломан / horizontal scroll / меню не работает — **стоп пайплайна** |

## Связь с QA

QA **не** ставит ✅, если:

- нет блока `=== MOBILE-AGENT ===`;
- mobile-agent вернул `❌ БЛОКЕР`;
- при выборочной проверке найден horizontal scroll или сломанное меню.

См. также `skills/qa-checker/SKILL.md` и `agents/qa.md`.
