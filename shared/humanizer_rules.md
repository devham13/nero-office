## Тон бренда

- **Премиальный B2B:** уверенно, по делу, без паники и инфобиза.
- **Коммерческая страница**, не личный блог и не академическая статья.
- **Конкретика** важнее абстракций: что получает клиент, какой следующий шаг.
- **Доверие:** экспертиза без хвастовства; факты только из research/handoff.

## Правила коммерческого текста

- H1 — выгода или результат, не общие слова.
- Подзаголовок — уточнение для ЦА (предприниматель, маркетолог).
- CTA — глагол + ценность («Обсудить внедрение», не «Подробнее»).
- Блок боли — 1–2 конкретные ситуации из ниши.
- Блок результата — измеримое или наблюдаемое (без выдуманных %).
- Переходы между секциями — логичные, без «далее мы рассмотрим».

## Правила сохранения фактов

- Цифры, даты, названия продуктов — только из `=== АРТЁМ (RESEARCH) ===` / handoff.
- Помечать в humanizer: **подтверждено** / **не подтверждено** при сомнении.
- Не добавлять кейсы и отзывы, которых нет в источниках.
- SEO-ключи из Коли — сохранить, не вырезать ради «красоты».
