## Твоя задача

Самостоятельно выбери одну лучшую тему для коммерческой страницы на основе данных из Google Таблицы:

https://docs.google.com/spreadsheets/d/10U4gH09L3uJqtxKdFM2_XH7LxRI39TNzEMGHWlCw-ls/edit?usp=sharing

Пользователь не обязан давать тебе тему вручную.

Ты сам:

1. читаешь Google Таблицу;
2. **берёшь только строки, где URL-колонка таблицы пустая** — если в ней уже есть URL, тема опубликована, строку **пропускай**;
3. **выбирай одну тему по очереди сверху вниз**: первая подходящая строка без ссылки (не «лучшую из десяти», а следующая в очереди);
4. из выбранной строки берёшь ключи, боль клиента, оффер, CTA, приоритет, сложность и ориентир чека;
5. проверяешь `<PROJECT_ROOT>/shared/kirill-news-ledger.md` и `<PROJECT_ROOT>/shared/published-pages.md`, чтобы не выбрать уже использованную тему;
6. используешь ключи из Google Таблицы как основной SEO-вход;
7. при необходимости проверяешь или расширяешь спрос через Wordstat;
8. записываешь результат в handoff для Коли и Артёма;
9. добавляешь выбранную тему в `<PROJECT_ROOT>/shared/kirill-news-ledger.md` со статусом `selected`.

**После публикации (Юра):** в Google Таблицу в URL-колонку **только для выбранной строки** записывается публичный URL (webhook из env). **Никогда не перезаписывай** уже заполненные ссылки в других строках.

Сравни кандидатов по скорингу из skill.

Отклони строки и дубли:

* **в колонке URL-колонка таблицы уже есть URL** (страница по этой теме опубликована);
* та же тема;
* тот же смысловой угол;
* похожий slug;
* уже опубликованная страница;
* уже выбранная тема в ledger.

Не предлагай список тем на выбор. Выбери одну тему, которая лучше всего даст трафик и лиды.
